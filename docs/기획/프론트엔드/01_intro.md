# 페이지 설계: `/intro` — 인트로

- 라우트: `/intro`
- 상태: 설계 확정 (구현 완료 — F1)

---

## 역할

서비스 첫 진입 시네마틱 경험. 사용자가 클릭해야 다음 단계로 진행.

---

## 레이아웃

- 풀스크린 (Header / MobileNav 없음)
- 배경: 검정

---

## 5단계 시퀀스

| 단계 | 내용 |
|------|------|
| 1 | 어두운 배경에 스타팅 포켓몬 실루엣 등장 (랜덤 5종: 피카츄/파이리/꼬부기/이상해씨/뮤츠) |
| 2 | 스캔 라인 애니메이션 (위→아래 sweep) |
| 3 | 실루엣 → 포켓몬 이미지 reveal |
| 4 | 텍스트 등장: "당신과 닮은 포켓몬이 있습니다" |
| 5 | CTA 버튼: "시작하기 →" 클릭 시 `/` 이동 |

- 자동 진행 없음 — 사용자 클릭으로만 단계 전환
- 건너뛰기 버튼: 우상단 "skip" → `/` 바로 이동

---

## 포켓몬 이미지 소스

`https://www.serebii.net/pokemongo/pokemon/{id}.png` (Pokemon GO 3D 아트)

랜덤 5종 ID: 25(피카츄), 4(파이리), 7(꼬부기), 1(이상해씨), 150(뮤츠)

---

## 구현 현황

F1 Stage에서 구현 완료. 현재 3단계 시퀀스로 동작 중.
설계 확정 버전(5단계)으로 업그레이드는 추후 F2 또는 F3에서 진행.
