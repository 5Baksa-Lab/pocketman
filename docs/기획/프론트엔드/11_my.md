# 페이지 설계: `/my` — 마이페이지

- 라우트: `/my`, `/my/edit`, `/my/password`
- 상태: 설계 확정 (F3에서 구현 예정)

---

## 역할

개인 계정 + 크리처 관리 허브. Instagram 프로필 구조 기반.

---

## `/my` — 메인 마이페이지

### 레이아웃

```
프로필 헤더
  - 아바타: 대표 크리처 이미지 (원형)
  - 닉네임
  - 한 줄 소개 (bio)
  - 통계: 크리처 수 / 좋아요 받은 수
  - [프로필 편집] 버튼

탭 영역
  [내 크리처] / [좋아요한 크리처]

크리처 그리드 (3열)
  - 공개: 🌍 배지
  - 비공개: 🔒 배지
  - 각 카드 클릭 → /creatures/{id}

설정 섹션
계정 섹션
```

## `/my/edit` — 프로필 편집

### 레이아웃

- 상단: 실시간 미리보기 카드 (davidkpiano 스타일)
  - 편집 내용이 카드에 즉시 반영됨
- 하단: 편집 폼

### 폼 필드

| 필드 | 제약 |
|------|------|
| 닉네임 | 중복 검사 필수 (실시간 debounce 300ms) |
| 한 줄 소개 | 최대 50자 |
| 대표 크리처 | 가로 스크롤 목록에서 선택 |

### 닉네임 중복 검사

```
입력 → debounce 300ms → GET /users/check-nickname?q={nickname}
응답 { available: true }  → 초록 체크 표시
응답 { available: false } → "이미 사용 중인 닉네임입니다" 인라인 에러
```

## `/my/password` — 비밀번호 변경

### 폼 필드

- 현재 비밀번호 확인
- 새 비밀번호 (8자 이상, 영문+숫자, 강도 인디케이터)
- 새 비밀번호 확인

### 예외 처리

- 소셜 로그인 전용 계정 (password = NULL): 섹션 숨김 + "소셜 로그인 계정은 비밀번호가 없습니다" 안내

## 설정 섹션

### 다크 모드

- 토글 스위치
- 기본값: 시스템 설정 (`prefers-color-scheme`) 따름
- 전환 효과: `transition: background-color 0.3s ease`
- 저장: `localStorage` + 서버 DB 동기화 (`PATCH /users/me { dark_mode: boolean }`)
- 전환 방식: `html.dark` 클래스 토글 + Tailwind `dark:` 변형

### 글자 크기 조절

| 단계 | font-size |
|------|----------|
| 작게 | 14px |
| 보통 (기본) | 16px |
| 크게 | 18px |

- 슬라이더 조작 시 즉시 미리보기 적용
- `html { font-size }` 변경으로 전체 적용 (rem 기반)
- 저장: `localStorage` + 서버 DB 동기화

## 계정 섹션

### 회원 탈퇴

2단계 확인:
1. 경고 모달: "탈퇴 시 모든 데이터가 즉시 삭제됩니다. 복구 불가."
2. 비밀번호 입력 (또는 소셜 계정: "탈퇴합니다" 텍스트 직접 입력)

탈퇴 처리:
- `DELETE /users/me`
- 즉시 삭제: 크리처 + 댓글 + 좋아요 기록 + 계정 정보 전체
- 세션 종료 → 랜딩 `/`으로 이동

---

## API 연동

- `GET /auth/me` → 프로필 데이터
- `PATCH /users/me` → `{ name?, bio?, avatar_creature_id? }`
- `PATCH /users/me/password` → `{ current_password, new_password }`
- `DELETE /users/me`
- `GET /users/check-nickname?q={nickname}` → `{ available: boolean }`
- `GET /creatures/my` → 내 크리처 목록
- `GET /creatures/liked` → 좋아요한 크리처 목록
