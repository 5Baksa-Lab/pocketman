## 전체 라우트 구조

```
/                       랜딩 페이지
/intro                  인트로 (시네마틱 애니메이션)
/login                  로그인
/signup                 회원가입

/upload                 얼굴 사진 업로드
/match                  포켓몬 매칭 결과 TOP 3
/generate/[id]          크리처 생성 대기
/result/[id]            크리처 결과 (첫 공개)

/plaza                  인터랙티브 광장 (멀티플레이어)
/creatures/[id]         크리처 상세 (공개 페이지)

/my                     마이페이지
/my/edit                프로필 편집
/my/password            비밀번호 변경
```

## 공통 레이아웃 구성요소

- **Header**: 모든 페이지 공통 (풀스크린 페이지 제외 — `/intro`, `/generate/[id]`)
- **MobileNav**: lg 미만 하단 고정 탭바 (홈 / 광장 / 만들기)
- **Sidebar**: lg 이상 좌측 네비 (F3에서 광장 개선 시 통합)

## 공통 컴포넌트

### PocketmanLogo

- 폰트: `Luckiest Guy` (Google Fonts)
- 색상: `#FFCE00` + `-webkit-text-stroke: 3px #1A3A6A`
- `paint-order: stroke fill`
- 3D 그림자: `text-shadow` 5레이어 누적

```css
.pocketman-logo {
  font-family: 'Luckiest Guy', cursive;
  font-size: clamp(32px, 6vw, 80px);
  color: #FFCE00;
  -webkit-text-stroke: 3px #1A3A6A;
  paint-order: stroke fill;
  text-shadow:
    2px 2px 0 #1A3A6A, 3px 3px 0 #1A3A6A,
    4px 4px 0 #1A3A6A, 5px 5px 0 #1A3A6A,
    6px 6px 0 #003A70,
    7px 7px 0 rgba(0, 0, 0, 0.25);
  letter-spacing: 4px;
  text-transform: uppercase;
}
```

사이즈 프롭: `sm`(24px) / `md`(40px) / `lg`(64px) / `xl`(80px)

참조 CodePen: `branneman/BaQBrL` (Pokemon logo CSS 기법)

## 기술 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | Next.js 15 (App Router) |
| 스타일 | Tailwind CSS |
| 인증 | NextAuth.js (Auth.js v5), JWT 전략 |
| 상태관리 | sessionStorage + URL 파라미터 (전역 상태 없음) |
| 폰트 | Luckiest Guy (로고), Lato (카드 정보) |
| 게임 엔진 | Phaser.js (광장 맵) |
| 실시간 | socket.io (WebSocket, 광장 멀티플레이어 + 채팅) |
| 컨페티 | canvas-confetti |
| DB | Railway PostgreSQL (기존 연결 유지) |
