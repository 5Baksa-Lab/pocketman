# Pokéman 프로젝트 로드맵 및 마일스톤 (3주 전략)

**작성일:** 2026년 3월 5일
**작성자:** 시니어 PM / 서비스 기획자
**전략 요약:** **"코어(CV)는 완벽하게, 확장은 영리하게(Demo Fallback)"**

이 로드맵은 3주라는 극단적인 타임라인 속에서 프로젝트의 '완성도'를 지키기 위한 전략적 마일스톤입니다. 1차 MVP(CV 코어)는 100% 실제 동작하도록 구현하며, 2차/3차 기능은 시간 부족 시 프론트엔드 UI 중심의 '데모(Mock-up)'로 전환할 수 있는 플랜 B를 내포하고 있습니다.

---

## 🚩 Phase 1: 1차 MVP - '핵심 가치 증명' (1주 차 완료 목표)
**목표:** 프로젝트의 기술적 근간인 CV 파이프라인의 완성도를 극대화하여 기술 심사를 완벽히 방어합니다.

*   **구현 상태:** **100% Backend & Frontend 연동 완벽 구현 필수**
*   **핵심 과제:**
    *   사용자 사진 업로드 및 유효성 검증 (얼굴 인식 실패 시 예외 처리)
    *   MediaPipe/OpenCV 기반 정밀한 얼굴 특징점(랜드마크) 추출
    *   추출된 데이터를 캐릭터 속성(타입, 형태)으로 매핑하는 알고리즘 확정
    *   이미지 생성 API(Stable Diffusion 등) 및 LLM 텍스트 스토리 생성 연동
*   **PM의 강조:** 이 단계에서 서버가 터지거나 결과물이 이상하게 나오면 프로젝트는 실패입니다. **전체 리소스의 70%를 여기에 쏟아부어야 합니다.**

## 🚩 Phase 2: 2차 MVP - '멀티모달 프로필 (스토리북)' (2주 차)
**목표:** 생성된 결과물에 생명력(음성, 동적 요소)을 불어넣어 와우 포인트(Wow Factor)를 만듭니다.

*   **구현 상태:** **실제 구현을 목표로 하되, 실패 시 'UI 데모 연출'로 전환 (Fallback)**
*   **Plan A (실제 구현):**
    *   프론트엔드 단에서 LLM이 생성한 대본을 감성적인 타이핑 애니메이션 자막으로 렌더링.
    *   이미지의 가벼운 CSS 애니메이션(줌인/아웃)과 자막 타이핑 효과를 동시 재생하여 스토리북 느낌 연출.
*   **Plan B (데모 우회 전략 - 시간이 없을 경우):**
    *   (TTS는 비범위로 확정되었으므로 해당사항 없음. 자막 애니메이션으로 충분한 연출 가능)
    *   만약 자막 타이핑 연출조차 시간이 부족하다면, 정적 이미지에 예쁜 폰트의 자막을 고정으로 박아넣는 수준으로 타협합니다.

## 🚩 Phase 3: 3차 MVP - '안전한 소통 광장 (SNS)' (3주 차)
**목표:** 사회적 고립을 해소한다는 프로젝트의 궁극적인 '사회적 이익' 비전을 시각적으로 증명합니다.

*   **구현 상태:** **프론트엔드 UI 데모(Mock-up) 우선 구현 후, 백엔드 연동**
*   **Plan A (실제 구현):**
    *   유저 크리처 정보를 담는 DB 테이블 설계 및 CRUD API 구현.
    *   익명 도감 피드 출력 및 이모지 리액션 카운트 업데이트 로직 구현.
*   **Plan B (데모 우회 전략 - 시간이 없을 경우):**
    *   복잡한 관계형 DB와 소셜 API 로직을 과감히 버립니다.
    *   프론트엔드에서 미리 만들어둔 **Mock JSON 데이터**를 활용해 인스타그램 피드처럼 예쁘게 꾸며진 '포켓 도감' 화면만 퍼블리싱합니다.
    *   리액션(💖) 버튼 클릭 시 화면상에서 숫자만 올라가게(상태값만 변경) 처리하여 "이런 식의 힐링 SNS로 동작한다"는 기획 의도만 완벽하게 전달합니다.

## 💡 결론 및 마인드셋
이 프로젝트의 성패는 **"CV 기술로 개인화된 결과물을 얼마나 납득 가능하게 뽑아내는가(Phase 1)"**에 달려 있습니다. 
멀티모달과 SNS 기능(Phase 2, 3)은 훌륭한 포장지입니다. 시간이 허락하는 한 최선을 다해 개발하되, 막힌다면 백엔드에서 시간 낭비하지 말고 **"가장 빠르고 아름답게 프론트엔드에서 동작하는 척(Demo)하게 만드는 것"**이 진짜 실력 있는 엔지니어링이자 해커톤 전략입니다.
