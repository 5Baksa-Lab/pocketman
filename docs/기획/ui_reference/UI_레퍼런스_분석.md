## 1. X (Twitter)

### 1-1. 랜딩 페이지 (비로그인)

**캡처 파일**: `x_landing.png`, `x_desktop_1280.png`, `x_mobile_390.png`

**데스크톱 레이아웃 (1280px)**:
```
┌────────────────────────────────────────────────────┐
│  [X 로고 / 좌측 절반]  │  [CTA 영역 / 우측 절반]   │
│                        │                           │
│  대형 X 로고 중앙 배치  │  "지금 일어나고 있는 일"  │
│  (좌측 50% 공간 차지)  │  "지금 가입하세요."        │
│                        │                           │
│                        │  [Google 계정으로 가입]    │
│                        │  [Apple에서 가입하기]      │
│                        │  ─────── 또는 ──────       │
│                        │  [계정 만들기] (검정 버튼) │
│                        │                           │
│                        │  "이미 가입하셨나요?"      │
│                        │  [로그인] (아웃라인 버튼)  │
│                        │  [Grok 다운로드]           │
├────────────────────────────────────────────────────┤
│  소개 | X앱 다운로드 | Grok | ... (하단 푸터 링크)  │
└────────────────────────────────────────────────────┘
```

**모바일 레이아웃 (390px)**:
- 상단 좌측: X 로고 (작게)
- 히어로 텍스트: 초대형 볼드 ("지금 일어나고 있는 일")
- CTA 버튼: 풀 width, 둥근 모서리(border-radius: 9999px)
- 2차 CTA: "이미 가입하셨나요?" + 로그인 버튼
- 하단: 푸터 링크 (wrap)

**핵심 UI 패턴**:

| 요소 | 스타일 |
|------|--------|
| 배경 | 순백 (#ffffff) |
| Primary CTA | 검정 배경 (#000000) + 흰 텍스트, 풀 rounded |
| Secondary CTA | 흰 배경 + 검정 테두리 + 검정 텍스트, 풀 rounded |
| Social Login | 아웃라인 버튼 + 아이콘 좌측 |
| 히어로 텍스트 | font-weight: 900, 매우 큰 사이즈 (데스크톱: ~64px) |
| 레이아웃 분할 | 정확히 50/50 (로고 영역 / CTA 영역) |

| 타이포 | 본문 14-15px, username bold |
| 미디어 | 카드 전체 너비, 최대 높이 제한 |

## 4. 3개 플랫폼 비교 요약

### 4-1. 레이아웃 구조 비교

| | X | Threads | Instagram |
|---|---|---|---|
| **데스크톱 레이아웃** | 3컬럼 (사이드바/피드/추천) | 3컬럼 (아이콘사이드/피드/패널) | 3컬럼 (사이드바/피드/추천) |
| **사이드바 스타일** | 좌측 고정, 텍스트+아이콘 | 좌측 고정, 아이콘만 (60px) | 좌측 고정, 텍스트+아이콘 (220px) |
| **모바일 네비** | 하단 탭바 | 하단 탭바 5개 | 하단 탭바 5개 |
| **랜딩 레이아웃** | 2컬럼 50/50 | 피드 직접 노출 | 2컬럼 50/50 |

### 4-2. 공통 패턴 (Pocketman에 반드시 적용)

1. **좌측 아이콘 사이드바** (lg 이상): 아이콘 중심, 텍스트는 xl 이상에서만
2. **하단 탭바** (md 이하): 5개 이내 주요 메뉴
3. **피드 카드**: 아바타 + 이름 + 시간 + 콘텐츠 + 액션 행
4. **CTA 버튼**: Primary(채움) / Secondary(아웃라인) 명확한 위계
5. **랜딩 2컬럼**: 좌=비주얼/우=액션 (또는 좌=비주얼+캐치프레이즈 / 우=CTA폼)
6. **순수 흰색 배경**: 불필요한 색상 배경 없음, 콘텐츠로 집중
7. **둥근 아바타**: 원형 (48px 피드, 32px 헤더)
8. **액션 아이콘 그룹**: 가로 나열, 텍스트 카운트 함께 표시

### 4-3. Pocketman 광장 피드에 적용할 패턴

**Threads 피드 구조를 주요 레퍼런스로 채택**:

```
이유:
1. Threads는 텍스트+미디어 혼합 카드 → Pocketman 크리처 카드와 유사
2. 비로그인 상태에서도 피드 공개 → Pocketman 광장과 동일 컨셉
3. 좌측 아이콘 사이드바 (60px) → 최소 공간으로 네비게이션 제공
4. 모바일 하단 탭바 5개 → 명확한 패턴
5. 반응 액션 아이콘 구조 → 이모지 리액션에 적용 가능
```

**Pocketman 광장 크리처 카드 적용 안**:
```
┌──────────────────────────────────────────────┐
│  [크리처 이미지 (Imagen 생성)]                │
│  ─────────────────────────────────────────── │
│  크리처 이름     [포켓몬 타입 배지]           │
│  스토리 텍스트 (2줄 말줄임)                   │
│  [영상 미리보기 thumbnail (있을 경우)]        │
│  ─────────────────────────────────────────── │
│  🔥 N  ❤️ N  👏 N  😂 N        [N일 전]    │
└──────────────────────────────────────────────┘
```
