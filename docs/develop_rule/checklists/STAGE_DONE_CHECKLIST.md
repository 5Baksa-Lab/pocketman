# Stage 완료 체크리스트

아래 항목을 모두 체크한 후에만 Stage 완료로 보고한다.

- [ ] Stage 목표 기능이 요구사항대로 동작한다.
- [ ] 코드 구조가 계층 규칙(router/domain/repository/adapter)을 준수한다.
- [ ] 핵심 실패 케이스(예외/타임아웃/빈값)를 처리했다.
- [ ] 테스트를 실행했고 결과를 기록했다. (미실행 시 사유 기록)
- [ ] 변경 파일 목록과 영향 범위를 정리했다.
- [ ] 남은 리스크/기술부채를 명시했다.
- [ ] 다음 Stage 인수인계 항목을 작성했다.
- [ ] 개발일지 파일을 생성했다.
  - [ ] 경로: `docs/development_logs/<part>/stage-<stage>/DEVLOG.md`
- [ ] PM/리드 리뷰를 받았다.
